
| Numerical Value | Equivalent Named Values |
|----------------:|-------------------------|
|               0 | emergency, emerg, panic |
|               1 | alert                   |
|               2 | critical, crit          |
|               3 | error, err              |
|               4 | warning, warn           |
|               5 | notice                  |
|               6 | informational, info     |
|               7 | debug                   |

Both numerical and named values are equally valid
